# Climate-Variability-and-Profitability-Analysis-of-Rice-Production-in-West-Bengal-Using-Power-BI
Climate Variability and Profitability Analysis of Paddy Production in West Bengal
Project Overview

This project analyzes paddy (rice) production trends in West Bengal by examining the relationship between climate variability, production levels, and economic profitability. Using historical agricultural and rainfall datasets, the analysis highlights how factors such as rainfall patterns, seasonal variations, and district-level differences influence rice production and farm profitability.

The project uses Power BI to build interactive dashboards that provide insights into production trends, cost–revenue dynamics, seasonal patterns, and climate impact on paddy cultivation.

Objectives

Analyze rice production trends over time in West Bengal

Evaluate the profitability of paddy cultivation by comparing revenue and production cost

Understand season-wise rice production patterns

Identify top and bottom performing districts in terms of production

Examine the impact of rainfall variability on rice production

Tools and Technologies

Power BI – Data visualization and dashboard development

Microsoft Excel / CSV datasets – Data preparation

Data Analytics techniques – Data cleaning, aggregation, and analysis

Data Sources

The analysis uses publicly available datasets:

Rainfall Dataset – India Rainfall Meteorological Subdivisions

Agricultural Crop Production Dataset – India Crop Production Statistics

Minimum Support Price (MSP) Dataset – India Data Portal

Dashboard Features

The Power BI dashboard includes:

Production Overview – Total production, revenue, cost, and profit trends

Profitability Analysis – Revenue vs cost comparison across districts

District-Level Analysis – Top and bottom producing districts with map visualization

Seasonal Production Analysis – Comparison of rice production across seasons

Rainfall Impact Analysis – Understanding how rainfall variability affects production

Key Insights

Rice production in West Bengal shows a gradual upward trend over the study period.

A few districts contribute a large share of total rice production.

Seasonal patterns strongly influence rice cultivation, with production concentrated during favorable climatic periods.

Rainfall variability affects production levels, highlighting the importance of climate conditions in paddy farming.

Despite increasing cultivation costs, paddy farming remains economically profitable in many districts.

Project Outcome

This project demonstrates how data analytics and visualization tools can help understand agricultural trends, support data-driven decision making, and highlight the impact of climate variability on crop production.
